🎯 2. framer-motion
🔥 Best for animations in React (replaces GSAP for most use cases).

Clean syntax for enter/exit transitions and scroll animations.

bash
Copy
Edit
npm install framer-motion
jsx
Copy
Edit
import { motion } from 'framer-motion';

<motion.div initial={{ opacity: 0 }} animate={{ opacity: 1 }} transition={{ duration: 1 }}>
  Hello world!
</motion.div>

⌨️ 5. react-hook-form
⚙️ Simplifies form state, validation, and error handling.

bash
Copy
Edit

jsx
Copy
Edit
import { useForm } from 'react-hook-form';

const { register, handleSubmit } = useForm();
<form onSubmit={handleSubmit(data => console.log(data))}>
  <input {...register("email")} />
  <button type="submit">Submit</button>
</form>

🔍 7. react-intersection-observer
Detect when elements enter or leave the viewport (great for triggering animations or lazy loading).

bash
Copy
Edit
npm install @react-intersection-observer
jsx
Copy
Edit
import { useInView } from 'react-intersection-observer';

const { ref, inView } = useInView();
<div ref={ref}>{inView ? 'Visible' : 'Not visible'}</div>
